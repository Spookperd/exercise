# THE ESSENTIALS PROGRAM - JEFF NIPPARD

**3x/Week • 45-Minute Workouts • 12-Week Program**

---

## PROGRAM OVERVIEW

This is a 12-week hypertrophy-focused program with three workout types that cycle:
1. **Full Body**
2. **Upper**
3. **Lower**

Each workout has multiple variations that rotate every 4 weeks to keep training fresh while allowing adequate progression time for each movement.

### Key Features:
- Time-efficient 45-minute workouts
- RPE-based autoregulation for flexible progression
- Exercise substitutions provided for equipment availability
- Dropsets and supersets for volume efficiency
- 1-2 rest days suggested between workouts

## PROGRESSION GUIDELINES

### How to Progress:
1. **Add reps** - Increase reps within the prescribed range while keeping weight the same
2. **Add weight** - Once you reach the top of the rep range, add weight and return to bottom of range
3. **Improve technique** - Better tempo control, better mind-muscle connection
4. **Improve mind-muscle connection** - Squeeze the target muscle harder

### Rep Range Progression Example:
- Week 1: 8 reps @ 100 lbs
- Week 2: 9 reps @ 100 lbs
- Week 3: 10 reps @ 100 lbs
- Week 4: 8 reps @ 105 lbs (added weight, back to bottom of range)

## RPE (RATE OF PERCEIVED EXERTION)

### RPE Scale:
| RPE | RIR (Reps in Reserve) | Description |
|-----|----------------------|-------------|
| 10 | 0 | Maximal Effort |
| 9.5 | 0 | No RIR, but could increase load |
| 9 | 1 | 1 RIR |
| 8.5 | 1-2 | Definitely 1, maybe 2 RIR |
| 8 | 2 | 2 RIR |
| 7.5 | 2-3 | Definitely 1, maybe 3 RIR |
| 7 | 3 | 3 RIR |
| 5-6 | 4-6 | 4-6 RIR |
| 3-4 | Many | Light Effort |
| 1-2 | Many | Light to no Effort |

### How to Use RPE:
- Select a weight that allows you to hit the prescribed RPE on the **LAST SET**
- Your first couple sets will be easier than the target RPE (this is normal)
- For example, if prescribed RPE is 8, you should have 2 reps left in the tank on your last set

## AUTOREGULATION

### What is Autoregulation?
Adjusting training on the fly based on daily performance rather than following a fixed program.

### How to Apply:
- **Strong days**: Push heavier than usual within RPE ranges
- **Weak days**: Use lighter loads that match your current abilities
- **Key principle**: Always maintain the prescribed RPE ranges

### Benefits:
- Allows for natural day-to-day performance variation
- Prevents overtraining on bad days
- Maximizes gains on good days
- More sustainable long-term

### Important Note:
Autoregulation doesn't mean training light all the time. You still need to push yourself and maintain accountability for overall progression.

## EXERCISE SELECTION NOTES

### Why These Exercises?
Since this is a hypertrophy program focused on time efficiency:
- **No Barbell Squat, Bench Press, or Deadlift** - These take significant time within a workout
- **Emphasis on dumbbells and machines** - More time-efficient and easier to swap out
- **Built-in flexibility** - Almost all exercises are swapped every 4-week block
- **Full 4 weeks to progress** - Adequate time to improve on each movement

### Exercise Substitutions:
- All substitutions work the same muscle groups with similar movement patterns
- No need to adjust sets, reps, RPE, or rest when substituting
- Simply perform the substitution exercise in place of the original

## PROGRAM PERIODIZATION

### The Split:
The program uses different splits based on training frequency:
- **3x/week (this version)**: Full Body / Upper / Lower split
- **4x/week version**: Upper / Lower split
- **5x/week version**: Upper / Lower / Push / Pull / Legs split
- **2x/week version**: Full Body split

### Block Structure:
- Each 4-week block features different exercise selections
- **Block 1 (Weeks 1-4)**: First set of exercises
- **Block 2 (Weeks 5-8)**: Second set of exercises
- **Block 3 (Weeks 9-12)**: Third set of exercises

### Volume:
- Lower volume per session compared to higher frequency programs
- Increased intensity/effort (RPE) to compensate
- Most exercises pushed close to failure

## KEY TERMS

| Term | Definition |
|------|------------|
| **CONCENTRIC** | The contracting ("positive") aspect of the lift |
| **DB** | Dumbbell |
| **DOMS** | Delayed onset muscle soreness |
| **ECCENTRIC** | The lowering ("negative") aspect of the lift |
| **EFFORT** | How hard you are pushing the set relative to failure. Measured with RPE and/or %1RM |
| **FREQUENCY** | How often you directly train a given muscle or lift every seven days |
| **HARD SET** | A set taken within 0-3 reps of failure (a "working set" and not a warm-up set) |
| **HYPERTROPHY** | The growth of (muscle) tissue |
| **INTENSITY** | Effort and load |
| **LOAD** | The weight of the external resistance |
| **LSRPE** | Last set RPE |
| **PERIODIZATION** | The organization of training over time |
| **PROGRESSIVE OVERLOAD** | The gradual increase of stress placed upon the body during exercise training. In training contexts, this generally involves progressively increasing some lifting parameter over time (usually increasing weight/reps or improving technique/mind-muscle connection) |
| **ROM** | Range of motion |
| **RPE** | Rate of perceived exertion. A measure of how difficult a set was on a 1-10 scale, with 10 meaning muscular failure was achieved. An RPE of 9 means you could have gotten one more rep, an RPE of 8 means you could have gotten two more reps, etc. |
| **TEMPO** | The speed at which the lift occurs |
| **VOLUME** | Total amount of work performed. Usually approximated as tough working sets |

## WARM-UP PROTOCOL

### General Warm-up (5-10 minutes):
- Light cardio (bike, treadmill, rowing)
- Dynamic stretching
- Joint mobility work

### Specific Warm-up:
Follow the prescribed warm-up sets in the tables:
- Start with lighter weights
- Gradually build to working weight
- Don't take warm-up sets to failure
- Rest 1-2 minutes between warm-up sets

### Example Warm-up for Heavy Exercise:
If your working weight is 225 lbs for 4-6 reps:
- Set 1: 135 lbs x 5 reps
- Set 2: 185 lbs x 3 reps
- Set 3: 205 lbs x 1 rep
- Working Set: 225 lbs x 4-6 reps @ RPE 8-9

## SUPERSET NOTATION

Exercises labeled **A1** and **A2** are performed as supersets:
1. Perform A1 exercise
2. Rest 0 minutes (or as prescribed)
3. Immediately perform A2 exercise
4. Rest the prescribed time for A2
5. Repeat for all sets

### Example:
- **A1: EZ Bar Skull Crusher** - 1 set, rest 0 min
- **A2: EZ Bar Curl** - 1 set, rest 1.5 min
- Repeat for 2 total rounds

## TRAINING TIPS

### Dropsets:
When a dropset is prescribed:
1. Perform the set to the prescribed RPE
2. Immediately reduce weight by ~50%
3. Continue for an additional 10-15 reps
4. Both portions count as a single dropset

### Form and Technique:
- Prioritize proper form over adding weight
- Control the eccentric (lowering) phase
- Full range of motion unless otherwise noted
- Maintain mind-muscle connection

### Rest Periods:
- Rest times are approximate
- Adjust based on recovery needs
- Heavier sets may need longer rest
- Lighter/isolation exercises can use shorter rest

### Progression Tracking:
- Log all workouts (weights, reps, RPE)
- Track week-to-week improvements
- Adjust based on performance trends
- Don't expect linear progress every session

## PROGRAM NOTES

### Time Efficiency:
- Workouts designed for ~45 minutes
- Barbell Squat, Bench, Deadlift excluded to save time
- Emphasis on dumbbells and machines for quick transitions
- Supersets and dropsets used for volume efficiency

### Equipment Needs:
- Dumbbells
- Cable machines
- Basic machines (leg press, machine chest press, etc.)
- Barbells (for some exercises)
- Adjustable bench

### Who This Is For:
- Intermediate to advanced trainees
- Those with limited time (45 min sessions)
- Anyone looking for a 3x/week hypertrophy program
- People who understand RPE-based training

### Who This Might Not Be For:
- Complete beginners (learn RPE first)
- Powerlifting-specific training
- Those wanting more training frequency
- People without access to basic gym equipment
